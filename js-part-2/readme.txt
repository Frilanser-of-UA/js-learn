=================== Массивы ===============
1) Добавление
   arr[1] = 50; Добавит в массив под индексом 1.
   arr[arr.lenght] = 50; Добавит в конец масива.
   arr.push(50,22); Добавит в конец масива. Вернет новую длинну.
   arr.unshift(99); Добавит в начеле массива. Вернет новую длинну.
2) Удаление
   arr.pop(); Удалит последний елемент с масива. Вернет удаленный елемент.
   arr.shift(); Удалит в начеле массива. Вернет удаленный елемент.
3) Обединяет массив в строку через черточку
   arr.join('-');
4) Проверка масив ли ето
   Array.isArray()
5) Обращение к елементу по индексу
   arr[2];
   arr.at(2);
6) Есть ли значение в масиве елемент
   arr.includes(66); // вернет  true/false
   arr.indexOf(66);  // вернет -1 если нету
7) arr.split('-') Метод split разбивает строки в массив;
   строка.split([разделитель], [максимальное количество элементов]);
8) Создание нового масива map()
   const array1 = [1, 4, 9, 16];
   map1 = array1.map((x) => x * 2);
   console.log(map1); Получим [2, 8, 18, 32]
9) Метод фильтрации масива filter()
10) Метод some находит первое совпадение в массиве возвращает true/false
11) Метод every находит все совпадение в массиве возвращает true/false
12) Метод find находит первое совпадение в массиве  Вернет сам елемент
13) arr.flat(); превратит двухмерный массив в одномерный
14) const arr_01 = Array(5); создает новый пустой массив  на 5 значений.
15) arr.fill(0, 2, 4) наполняет / перезаписивает значения в массиве на 0 от 2го до 3го значения
